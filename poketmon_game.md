# 7살 도형 IQ 게임 설계서

> 목적: **스토리 없이 도형 추론 능력을 기르는 7살 수준 IQ 게임**을 구현한다.
>
> 특징:
> - 게임의 도형은 public/poketmonster 하위의 이미지를 사용한다.
> - 규칙은 항상 **1개만 적용**
> - 텍스트 설명 최소화 (비언어적 추론)

---

## 1. 타겟 연령 및 설계 원칙

### 🎯 대상
- 만 7세 (초등 입학 전후)

### ✅ 설계 원칙
- 규칙 1개만 사용
- 시간 제한 없음
- 실패 패널티 없음
- 시각적으로 즉시 이해 가능
- bgm은 public/sounds폴더하위의 bgm.mp3를 사용한다

### ❌ 제외 요소
- 좌우 반전
- 규칙 2개 이상 혼합
- 블록 6칸 초과

---

## 2. 핵심 게임 유형 (총 8~12개 확장 가능)

본 설계서는 아래 **도형 수열 문제 유형**을 기준으로 한다.

---

## 3. 게임 유형 상세 설계

### 3.1 회전 규칙 게임 (Rotation)

#### 규칙
- 같은 도형이 **90도씩 회전**하며 등장

#### 예시 수열
```
도형 A → 도형 A (90°) → 도형 A (180°) → ?
```

#### 난이도
- Lv1: 2개 제시
- Lv2: 3개 제시
- Lv3: 4개 제시

#### 인지 자극
- 공간 지각
- 정신적 회전

---

### 3.2 블록 개수 변화 게임 (Add / Remove)

#### 규칙
- 도형을 이루는 블록 수가 **+1 또는 -1**

#### 제한
- 7살 기준: +1 / -1 만 허용

#### 인지 자극
- 수량 감각
- 패턴 예측

---

### 3.3 위치 이동 게임 (Translation)

#### 규칙
- 도형 모양은 고정
- 위치만 이동

#### 방향
- 좌 → 우
- 하 → 상

#### 인지 자극
- 시각 추적
- 주의 집중

---

### 3.4 색 위치 이동 게임 (Color Shift)

#### 규칙
- 색 블록의 위치만 변경

#### 제한
- 색상 2~3개만 사용

#### 난이도
- 초반: 2색
- 후반: 3색

---

### 3.5 부분 변화 게임 (Partial Change)

#### 규칙
- 전체 도형 중 **일부 블록만 변화**

#### 예시
- 파란 블록은 고정
- 빨간 블록만 이동

#### 난이도
- 7살 상위 레벨용

---

### 3.6 방향 전환 게임 (Orientation)

#### 규칙
- 도형의 위/아래 방향만 변경

> ⚠ 좌우 반전은 제외

---

## 4. 게임 구성 방식

### 📦 하나의 게임 = 하나의 규칙

- 규칙 1종
- 난이도 3단계

#### 예시
- 회전 게임
  - Lv1
  - Lv2
  - Lv3

➡ 총 **8~12개의 미니 게임** 구성 가능

## 5. React 구현 가이드

### 5.1 데이터 구조

```ts
type Cell = {
  x: number
  y: number
  color?: string
}

type Shape = {
  cells: Cell[]
  rotation: 0 | 90 | 180 | 270
}

type Rule =
  | 'ROTATE'
  | 'ADD_BLOCK'
  | 'MOVE'
  | 'COLOR_SHIFT'
  | 'PARTIAL_CHANGE'
```

## 6. 난이도 조절 기준

| 요소 | Lv1 | Lv2 | Lv3 |
|----|----|----|----|
| 제시 도형 수 | 2 | 3 | 4 |
| 색상 수 | 1 | 1~2 | 2~3 |
| 선택지 수 | 2 | 3 | 4 |
